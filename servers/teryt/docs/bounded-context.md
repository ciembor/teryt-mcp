# TERYT MCP bounded context

TERYT MCP covers official Polish territorial registry lookup and synchronization.

## Included

- TERC.
- SIMC.
- ULIC.
- WMRODZ.
- Synchronization of official TERYT data.
- Search and address resolution up to street level.

## Excluded

- REGON.
- BDL.
- PRG.
- Geocoding.
- Maps.
- Unofficial address data.
